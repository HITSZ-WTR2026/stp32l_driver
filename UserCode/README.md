## 项目结构规划

```text
UserCode/
├── bsp/                      # 板级驱动，控制板上外设 (I2C, SPI, UART, CAN, ...)
├── drivers/                  # 驱动程序，驱动硬件
├── third_party/              # 第三方（比如厂家）提供的驱动库
├── libs/                     # 库文件，提供算法、功能逻辑、通用模块，不依赖具体硬件
├── interface/                # 接口层，用于对不同的外设提供统一的向上接口
├── controllers/              # 控制层，用于实现 外设+硬件
├── app/                      # 应用层
```

在同一层的文件之间**不会存在**相互引用关系
